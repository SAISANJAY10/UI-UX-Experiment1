### **Aim:**

To explore the fundamentals of UI and UX, differentiate between UI and UX using real-world case studies, understand their relationship in design, and demonstrate various UI/UX design tools through hands-on tasks.

---

### **Algorithm:**

1. Study the basic concepts of UI and UX.
2. Identify differences between UI and UX using examples.
3. Create a simple persona and user flow to illustrate UX.
4. Design basic wireframes to demonstrate UI layout.
5. Use UI/UX tools (Figma, Canva, Balsamiq, etc.) to create sample designs.
6. Analyze how UI and UX work together to improve product usability.
7. Document outputs and compile them into the lab report.
